## FAQ

### HKG.EB.WEv2 和 HKG.EB.TINY 怎么选？

主要看流量需求。WEEv2 月流量 600 GB，TINY 是 1200 GB，内存也从 1 GB 翻到 2 GB。如果你只是跑个轻量代理或者个人小项目，WEEv2 完全够用；如果有多个服务同时跑，或者流量消耗比较大，直接上 TINY 更省心，价格差距不算大。

### DMIT 香港 VPS 支持退款吗？

DMIT 提供 72 小时内无理由退款政策，新购套餐在 72 小时内如果不满意可以申请全额退款。超过这个时间窗口后不支持退款，所以买之前最好先想清楚需求。

### CN2 GIA 线路在晚高峰表现怎么样？

这是很多人最关心的问题。CN2 GIA 的核心优势就是晚高峰抗拥堵能力，相比普通 BGP 线路，晚高峰延迟增幅明显更小。当然没有任何线路能保证绝对不受影响，但 GIA 在同价位里确实是最稳的选择之一。

### DMIT HKG.EB 系列可以用来建站吗？

可以，KVM 虚拟化支持完整的 Linux 环境，跑 Nginx、Apache、数据库都没问题。WEEv2 的 1 GB 内存对于小型静态站或者低流量动态站够用，如果是有一定访问量的站点，建议至少选 TINY 或 Starter。

### 流量超了之后会怎样？

月流量用完后，DMIT 会对带宽进行限速，服务不会中断。限速后的带宽仍然可以正常使用，只是速度会降低。下个计费周期开始后流量自动重置，恢复正常带宽。

### DMIT 支持哪些操作系统？

支持主流 Linux 发行版，包括 Debian、Ubuntu、CentOS、AlmaLinux 等，可以在控制面板直接选择重装。不支持 Windows 系统。
